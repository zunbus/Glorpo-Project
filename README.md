2. Instrukcja uruchamiania
a) WYMAGANIA SPRZĘTOWE
Unreal Engine 5 polega na nowoczesnych systemach graficznych,
co oznacza, że sprzęt powinien być co najmniej przyzwoity.

Przybliżone minimalne wymagania sprzętowe:
-CPU: Intel i5 7xxx / AMD Ryzen 5 1600 lub równoważny
-RAM: 16 GB
-GPU: NVIDIA GTX 1060 6GB / AMD RX 580 8GB (obsługa DirectX12 lub Vulkan)
-Dysk: 700 MB wolnego miejsca
-System: Windows 10 (64-bitowy)

Gra wymaga bibliotek uruchomieniowych w systemie docelowym, głównie "Microsoft Visual C++ Redistributable", zazwyczaj są one instalowane automatycznie z systemem Windows.
Gra wymaga aktualnych sterowników dla karty graficznej zamontowanej w komputerze.

b) Rozpakowanie pliku .zip
Pobranie pliku o rozszerzeniu zip ze strony _____________ oraz rozpakowanie go na komputerze (Prawy przycisk myszy > wyodrębnij wszystkie) z wystarczającą ilością dostępnego miejsca.

c) Uruchomienie pliku .exe znajdującego się w folderze gry.
Może pojawić się ostrzeżenie Windows Defender. Należy zezwolić na uruchomienie.
Jeżeli gra ma problemy z uruchomieniem, zwrócić się do sekcji "WYMAGANIA SPRZĘTOWE".
Gra jest uruchamiana z najniższymi możliwymi ustawieniami graficznymi, można je zmienić wchodząc w menu pauzy (Esc) > Ustawienia.
